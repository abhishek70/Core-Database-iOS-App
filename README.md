Core-Database-iOS-App
=====================

Demonstrate the functionality of adding the contact data to the database and displaying the list of contacts on
swipe operation vertically.

Requirements:

Xcode 5.1.1 -
iOS SDK 7.1.1 -
SQLite Database Engine
